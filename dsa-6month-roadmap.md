## Daily Schedule (2 Hours)

```
Option 1: Single Block (Evening)
├─ 0:00-0:15  → Revision of yesterday's problem
├─ 0:15-0:45  → Learn pattern / Watch concept
├─ 0:45-1:30  → Solve 1-2 problems (untimed)
└─ 1:30-2:00  → Log problems, write pattern notes

Option 2: Split Blocks
├─ Morning (30 min) → Quick revision / Easy problem
└─ Evening (90 min) → Main learning + practice
```

## Red Flags to Watch

| Warning Sign | Fix |
|--------------|-----|
| Solved but forgot solution | Revisit after 3 days |
| Can't explain the logic | Write explanation in notes |
| Taking too long on Easy | Practice more Easy, build speed |
| Stuck on pattern for days | Move on, come back later |
| Missing practice days | Set phone alarm, same time daily |
